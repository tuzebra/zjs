ZJS core

Zjs là một thư viện javascript giúp thao tác với DOM một cách đơn giản, nhanh chóng và hiệu quả.

Sau khi nhúng vào trang web, sẽ có 2 đối tượng global tồn tại để sử dụng: zjs và z (z là shortcut của zjs)

Có thể dùng zjs theo nhiều cách khác nhau

  • Dùng để thao tác với DOM bằng cách tạo ra một zjsDOMObject

    // Tạo ra một zjsDOMObject
var a = zjs('css query string');

  • Dùng để bắt sự kiện domready và gọi hàm xử lý

    zjs(function(){			
    // code javascript trong đây 
    // sẽ được chạy khi domready
});

  • Dùng như một static class để gọi các method & property

    // Xem thông tin về Browser thông qua property browser
zjs.browser;

// Sử dụng ajax thông qua method ajax
zjs.ajax();

zjsDOMObject

Object chứa các method để thao tác với DOM

Một zjsDOMObject lưu trữ một hoặc nhiều đối tượng DOM, vì vậy nên có thể thao tác trên nhiều đối tượng DOM cùng lúc

Để tạo ra một zjsDOMObject

var a = zjs('#test1');             // tạo ra zjsDOMObject chứa 1 element có id là "test1"
var b = zjs('.test2');             // tạo ra zjsDOMObject chứa những element có class là "test2"
var c = zjs(window);               // tạo ra zjsDOMObject chứa đối tượng window
var d = zjs(a);                    // tạo ra zjsDOMObject chứa những element mà a (là một zjsDOMObject) chứa
var e = zjs([b, c]);               // tạo ra zjsDOMObject chứa những element mà b và c chứa
var f = zjs(['#test1', '.test2']); // tạo ra zjsDOMObject chứa những element có id là "test1" hoặc có class là "test2"
var g = zjs('#test1, .test2');     // tạo ra zjsDOMObject chứa những element có id là "test1" hoặc có class là "test2"

Để thao tác với những element chứa trong một zjsDOMObject ta sử dụng các method của zjsDOMObject

// tạo ra zjsDOMObject chứa những element có class là "test2"
var a = zjs('.test2');

// thay đổi nội dung (innerHTML) của toàn bộ những element nào có class là "test2"
a.html('nội dung mới...');

// một cách viết code ngắn hơn
zjs('.test2').html('nội dung mới...');

Các method của zjsDOMObject thường hỗ trợ việc gọi theo chuỗi liên tục

// thay đổi nội dung (innerHTML), sau đó thay đổi màu chữ
// của toàn bộ những element nào có class là "test2"
zjs('.test2').html('nội dung mới...').style('color', 'red');
