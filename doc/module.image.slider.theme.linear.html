Linear Theme

Các option khi sử dụng với phương thức zjs::makeSlider(object option)

theme string Tên của theme dùng làm slide, luôn luôn đặt như sau:
theme: 'linear'
transition int string Hiệu ứng chuyển giữa các hình, chấp nhận cả kiểu int và string.
mặc định: 
transition: 0
các giá trị có thể chấp nhận:
0 'none'không sử dụng hiệu ứng
1 'fade'hiệu ứng fadeIn
2 'horizontal'hiệu ứng trượt theo chiều ngang
3 'vertical'hiệu ứng trượt theo chiều dọc
transitionTime int Thời gian (milisecond) chạy hiệu ứng chuyển giữa các hình (nếu như có sử dụng).
mặc định: 
transitionTime: 1600
transitionTimingfunction int Quy định cách chạy của hiệu ứng.
mặc định: 
transitionTimingfunction: 2

các giá trị này là thiết lập của zjs.timer(), xem tại đây.

các giá trị có thể chấp nhận:

0 'linear'
1 'sinoidal'
2 'quadratic'
3 'cubic'
4 'elastic'
border bool Đặt là true nếu muốn xuất hiện đường viền xung quanh slide.
mặc định: 
border: true
preload bool Đặt là true nếu muốn xuất hiện phần trăm (%) load hình trước khi xuất hiện slide.
mặc định: 
preload: true

tuỳ chọn này yêu cầu phải có module image.loader, xem tại đây.
autoplay bool Đặt là true nếu như muốn slide tự động chuyển hình.
mặc định: 
autoplay: true

có thể sử dụng phương thức zjs::slidePlay() để bật chế độ cho slide tự động chuyển hình.
autoplayTime int Thời gian (milisecond) chờ giữa 2 hình trước khi chuyển.
mặc định: 
autoplayTime: 6000
playButton bool Đặt là true nếu muốn xuất hiện 2 button play, pause và thanh đếm lùi tới lúc chuyển hình.
mặc định: 
playButton: false

phương thức liên quan: zjs::slidePlay(), zjs::slidePause()
navButton bool Đặt là true nếu muốn xuất hiện 2 button next và previous.
mặc định: 
navButton: true
navButtonParentIsRoot bool Đặt là true nếu muốn 2 navigation button (button next, previous) đươc đặt ra bên ngoài (không nằm trong <div class="nav-container"> nữa).
mặc định: 
navButtonParentIsRoot: false
navThumb bool Đặt là true nếu muốn xuất hiện các ảnh thu nhỏ (thumbnail).
mặc định: 
navThumb: true
navDot bool Đặt là true nếu muốn xuất hiện các chấm tròn (tương tự như thumbnail).
mặc định: 
navDot: true
linkTitle bool Đặt là true nếu muốn title của từng slide có thể gắn được link.
mặc định: 
linkTitle: false
linkCover bool Đặt là true nếu muốn từng slide được gắn link (phạm vi click link trong toàn bộ slider).
mặc định: 
linkCover: false
sequent bool Đặt là true nếu muốn các slide được sắp xếp và di chuyển một cách có thứ tự.
mặc định: 
sequent: true
fullWidth bool Đặt là true nếu muốn chiều rộng của slider tự co dãn cho fix với kích thước của element cha (element chứa slider). Tuỳ chọn này thích hợp để làm với những web responsive design.
mặc định: 
fullWidth: false
fullscreenWidth bool Đặt là true nếu muốn chiều rộng của slider tự co dãn cho fix với kích thước màn hình. Tuỳ chọn này thích hợp để làm với những web responsive design.
mặc định: 
fullscreenWidth: false
hoverToSlide bool Đặt là true nếu muốn chỉ cần hover vào thumbnail là sẽ chuyển slide.
mặc định: 
hoverToSlide: false
navThumbTemplate string function Quy định template cho từng hình thumbnail.
mặc định: 
navThumbTemplate: '<a class="nav-image"></a>'

cũng có thể dùng function thay cho string: 
navThumbTemplate: function(index){
	return '<a class="nav-image"></a>'
}

trong đó index là vị trí của hình thumb hiện tại.
navDotTemplate string function Quy định template cho từng chấm tròn (tương tự như thumbnail).
mặc định: 
navDotTemplate: '<a class="nav-dot">.</a>'

cũng có thể dùng function thay cho string: 
navDotTemplate: function(index){
	return '<a class="nav-dot">'+index+'</a>'
}

trong đó index là vị trí của hình thumb hiện tại.

customElementsClass: 'elements'

xem thêm về custom elements và custom transition tại đây.

customTransitionClass: 'transition'

xem thêm về custom elements và custom transition tại đây.

Khởi tạo nhanh ngay trong code html với default option

chỉ cần đặt class zslider và option dùng theme linear là xong, không cần viết bất kỳ code javascript nào

html:

<div class="zslider" data-option="theme:'linear'">
    <ul>
        <li>
            <h3>title 1</h3>
            <img src="thumb1.jpg" largesrc="image1.jpg" />
            <p>Description 1</p>
        </li>
        <li>
            <h3>title 2</h3>
            <img src="thumb2.jpg" largesrc="image2.jpg" />
            <p>Description 2</p>
        </li>
    </ul>
</div>

Khởi tạo nhanh ngay trong code html kèm theo các option khác

đặt class zslider cho element, và kèm theo những option khác ngay trong code html

html:

<div class="zslider" data-option="theme:'linear', transition:'fade', transitionTime:2000, 
	transitionTimingfunction:'quadratic', autoplayTime:3000">

Khởi tạo bằng code javascript với các default option

html:

<div id="divdivdiv">
    <ul>
        <li>
            <h3>title 1</h3>
            <img src="thumb1.jpg" largesrc="image1.jpg" />
            <p>Description 1</p>
        </li>
        <li>
            <h3>title 2</h3>
            <img src="thumb2.jpg" largesrc="image2.jpg" />
            <p>Description 2</p>
        </li>
    </ul>
</div>

javascript:

zjs('#divdivdiv').makeSlider();

Khởi tạo bằng code javascript và tuỳ chỉnh các option

javascript:

zjs('#divdivdiv').makeSlider({
		theme:'linear',
		transition:'fade',
		transitionTime:2000,
		transitionTimingfunction:'quadratic',
		autoplayTime: 3000
	});

Khởi tạo bằng code javascript và các option được set ở cả html và javascript

html:

<div id="divdivdiv" data-option="theme:'linear', transition:'fade'">

javascript:

zjs('#divdivdiv').makeSlider({
		transitionTime:2000,
		transitionTimingfunction:'quadratic',
		autoplayTime: 3000
	});

Lắng nghe sự kiện khi chuyển sang 1 slide mới

javascript:

zjs('#divdivdiv').on('slider.change', function(event, element){
		console.log(event.data);
		console.log(element); // element #divdivdiv
	});

xem giải thích về sự kiện slider.change và các thuộc tính của event.data tại module.image.slider.html#onchange

Custom element và custom transition

Custom element

Mặc định thì khi 1 element được biến thành slider, thì toàn bộ cấu trúc html sẽ được thay đổi. Tuy nhiên nếu bạn vẫn muốn custom một số thành phần trong từng slide, hãy làm như sau:

html:

<div class="zslider" data-option="theme:'linear'">
    <ul>
        <li>
            <h3>title 1</h3>
            <img src="thumb1.jpg" largesrc="image1.jpg" />
            <p>Description 1</p>
            <div class="elements">
            	<a>custom</a>
            	<p>custom</p>
            </div>
        </li>
        <li>
            <h3>title 2</h3>
            <img src="thumb2.jpg" largesrc="image2.jpg" />
            <p>Description 2</p>
        </li>
    </ul>
</div>

Theo đoạn mã trên thì trong slide đầu tiên (thẻ li đầu tiên) tất cả những gì đặt bên trong thẻ <div class="elements"> sẽ được giữ lại mà không bị thay đổi một cách tự động. Vậy nên hãy đặt mọi thứ bạn muốn custom vào bên trong thẻ div với class là elements này.

Không muốn đặt tên class là elements? bạn có thể tuỳ chọn tên cho class này ở tuỳ chọn customElementsClass

Custom transition

Bạn muốn những custom elements trong từng slide có những hiệu ứng chuyển động cho đẹp mắt? Hãy sử dụng class transition

<div class="zslider" data-option="theme:'linear'">
    <ul>
        <li>
            <h3>title 1</h3>
            <img src="thumb1.jpg" largesrc="image1.jpg" />
            <p>Description 1</p>
            <div class="elements">
            	<a class="transition" data-option="">custom</a>
            	<p class="transition" data-option="">custom</p>
            </div>
        </li>
        <li>
            <h3>title 2</h3>
            <img src="thumb2.jpg" largesrc="image2.jpg" />
            <p>Description 2</p>
        </li>
    </ul>
</div>

Các hiệu ứng được thực hiện bởi module transition của zjs, nên bạn phải nhúng module này vào trước khi khởi tạo slider.

<script type="text/javascript" src="//app.april.com.vn/js/z.module.transition.js"></script>

(Nếu như dùng zjs.require thì không cần phải nhúng thủ công module transition này)

Các thiết lập hiệu ứng bên trong data-option là các thiết lập của module transition, xem tại module.transition.html

Không muốn đặt tên class là transition? bạn có thể tuỳ chọn tên cho class này ở tuỳ chọn customTransitionClass
