zjsDOMObject method

Một số quy ước

  • Trong tài liệu này quy ước zjs:: được hiểu như là một thể hiện của zjsDOMObject

    Tức là với cách viết zjs::html() thì html sẽ được hiểu là một method của zjsDOMObject, và để sử dụng được method html thì ta phải tạo ra một thể hiện của zjsDOMObject

    // tạo ra một thể hiện của zjsDOMObject
var a = zjs('css query string');
// sử dụng method html
a.html('bla bla');

  • Các method hầu hết là return về lại chính zjsDOMObject để có thể gọi liên tiếp theo kiểu như sau:

    zjs('css query string').html('bla bla').style('color', 'red');

  • Nếu trong tài liệu không đề cập đến return thì sẽ ngầm hiểu là method return về lại chính zjsDOMObject

  • Nếu method return về một kiểu dữ liệu khác, thì kiểu dữ liệu đó sẽ được đặt trước cú pháp của method, chẳng hạn như:

    string zjs::getValue() ta hiểu là method này sẽ return về một string

  • Nếu một method có nhiều cách sử dụng khác nhau, thì sẽ có nhiều cách truyền parameter khác nhau, trong tài liệu sẽ phân ra trình bày nhiều hơn 1 cú pháp cho method đó, nhằm dễ theo dõi.

    Ví dụ với method attr có 2 cách sử dụng khác nhau:

    string zjs::attr(string name) get attribute của element và return về một string

    zjs::attr(string name, string value) set attribute cho element và return về lại zjsDOMObject

  • Các method ngoài mục cú pháp còn có thể có thêm mục hook trình bày về những hook mà method đó có

style

Get thông tin style của element

cú pháp:

string zjs::style(string name)

name string tên của style muốn lấy thông tin

ví dụ:

// <a style="color:red; margin-left:10px; ">
var color = zjs('a').style('color'); // color = 'red'
var marginLeft = zjs('a').style('margin-left'); // marginLeft = '10px'

nếu muốn kết quả trả về là integer thay vì string, sử dụng cú pháp sau:

int zjs::style(string name, true)

// <a style="color:red; margin-left:10px; ">
var marginLeft = zjs('a').style('margin-left', true); // marginLeft = 10

Set một style cho element

cú pháp:

zjs::style(string name, mixed value)

name string tên của style muốn thay đổi
value string int giá trị (chấp nhận cả string hoặc integer)

ví dụ:

zjs('a').style('color', 'red');
zjs('a').style('margin-left', 10);

Set cùng lúc nhiều style cho element

cú pháp:

zjs::style(object styles)

styles object tập hợp các giá trị style muốn thay đổi

ví dụ:

zjs('a').style({
	color: 'red',
	margin: 10
});

ví dụ:

hasClass

kiểm tra class có tồn tại trên element hay không

cú pháp:

bool zjs::hasClass(string name)

name string class cần kiểm tra

ví dụ:

// <h1 class="normal red">
zjs('h1').hasClass('title'); // false
zjs('h1').hasClass('normal'); // true

setClass

thay thế chuỗi class hiện tại của element (những class đang tồn tại trên element sẽ bị mất)

cú pháp:

zjs::setClass(string name)

name string chuỗi class của element, có thể chỉ là 1 class, hoặc nhiều class (mỗi class cách nhau khoảng trắng)

ví dụ:

zjs('div').setClass('header');

toàn bộ những class hiện tại sẽ bị thay đổi:

// <h1 class="normal red">
zjs('h1').setClass('title bold uppercase'); // <h1 class="title bold uppercase">

addClass

khác với setClass, addClass không thay đổi những class hiện tại, mà chỉ thêm vào một hoặc nhiều class mới

cú pháp:

zjs::addClass(string name)

name string tên class, hỗ trợ nhiều tên class cùng lúc (cách nhau khoảng trắng)

ví dụ:

// <h1 class="normal red">
zjs('h1').addClass('title'); // <h1 class="normal red title">
zjs('h1').addClass('bold uppercase'); // <h1 class="normal red title bold uppercase">

removeClass

gỡ bỏ một hoặc nhiều class của element

cú pháp:

zjs::removeClass(string name)

name string tên class cần gỡ bỏ, hỗ trợ nhiều tên class cùng lúc (cách nhau khoảng trắng)

ví dụ:

// <h1 class="normal red title bold uppercase">
zjs('h1').removeClass('title'); // <h1 class="normal red bold uppercase">
zjs('h1').removeClass('bold uppercase'); // <h1 class="normal red">

toggleClass

chuyển đổi qua lại giữa 1 class

cú pháp:

zjs::toggleClass(string name)

name string tên class cần chuyển (không hỗ trợ nhiều tên class cùng lúc)

ví dụ:

// <h1 class="">
zjs('h1').toggleClass('normal'); // <h1 class="normal">
zjs('h1').toggleClass('normal'); // <h1 class="">
zjs('h1').toggleClass('normal'); // <h1 class="normal">

append

tạo ra element mới và append vào mọi element đang chứa trong zjsDOMObject

cú pháp:

zjs::append(string html)

html string mã html tạo ra element mới

ví dụ:

// <p>1</p> <p>2</p> 
zjs('p').append('<a>'); 
// <p>1<a></a></p> <p>2<a></a></p>

hỗ trợ truyền vào cùng lúc nhiều param

ví dụ:

// <p>1</p> <p>2</p>
zjs('p').append('<a>', '<b>'); 
// <p>1<a></a><b></b></p> <p>2<a></a><b></b></p>

append element đang có sẵn vào element đầu tiên chứa trong zjsDOMObject

cú pháp:

zjs::append(element el)

el element element đã tồn tại trong cây DOM

ví dụ:

// <p>1</p> <p>2</p> <a>3</a> <a>4</a>
// lấy ra element <a> đầu tiên
var a = zjs('a').item(0, true);
// append a vào <p> đầu tiên
zjs('p').append(a);
// <p>1<a>3</a></p> <p>2</p> <a>4</a>

append nhiều element đang có sẵn vào element đầu tiên chứa trong zjsDOMObject

cú pháp:

zjs::append(zjsDOMObject zobj)

zobj zjsDOMObject zjsDOMObject chứa những element cần append

ví dụ:

// <p>1</p> <p>2</p> <a>3</a> <a>4</a>
zjs('p').append(zjs('a'));
// <p>1<a>3</a><a>4</a></p> <p>2</p>
