please view html source to know how-to use the option

Default option

<input class="zuislider" type="hidden" value="0" />

Can set the min and max value

<input class="zuislider" type="hidden" value="0" data-option="min:1,max:100" />

Can set step

<input class="zuislider" type="hidden" value="0" data-option="min:1,max:100,step:1" />

<input class="zuislider" type="hidden" value="0" data-option="min:10,max:20,step:1" />

<input class="zuislider" type="hidden" value="12" data-option="min:10,max:20,step:3" />

<input class="zuislider" type="hidden" value="15" data-option="min:10,max:20,step:0.5" />

<input class="zuislider" type="hidden" value="1" data-option="min:1,max:12" />

You can set array of values you want to use on slider

<input class="zuislider" type="hidden" value="1" data-option="valueArray:[11,4,5,2,1,9,10,8,16]" />

You can use code to set the slider value

$('.zuislider').setValue(14)
